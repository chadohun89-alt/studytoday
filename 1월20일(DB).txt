* 제약조건 ==> 무결성(에러 없음)
1. primary key (기본 키)
2. Foreign key (참조 키, 외래키)
3. unique (중복 허용 안 함)
4. not null ( null 허용 안 함)
5. default ( 자동으로 값 채워 넣음)
6. enum("남자", "여자")
7. check (price > 0)

* 쿼리(querry)
select : 조회, 검색
insert : 삽입, 추가, 등록
update : 수정
delete : 삭제
 
 ----

order by (정렬)

select no, id, name from member order by no asc; (오름차순)
select no, id, name from member order by no desc; (내림차순)
select no, id, name from member order by name asc, id desc; (내림차순)


update(수정)

update member
set name = '홍성순' 
where no = 3;

update member set name = '이성순' where no =3;

* SQL 명령어 분류

DDL   : 데이터 정의어    - create, alter(비밀번호 등 변경), drop, truncate
DML  : 데이터 조작어    - select, insert, update, delete > record 대상
DCL  : 데이터 제어어     - grant, revoke
TCL  : 트랜잭션 제어어   - commit, rollback, savepoint
(물리적으로 2개의 동작인데 논리적으로 1개의 동작)
